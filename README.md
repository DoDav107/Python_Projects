CITS1401 Python Project 1 - CSV File Reader and Data Processing

This Python project is a CSV file reader and data processor developed as part of the CITS1003 unit at the University of Western Australia. Its primary function is to read data from CSV files, process it into a structured format using dictionaries and lists, and provide easy access to the data for further analysis.

Project Overview

The project focuses on:

Reading CSV Files: Uses the open() function to read CSV files and extract headers and data. Data Structuring: Converts each row into a dictionary using headers as keys. Error Handling: Implements try-except blocks to manage file reading errors gracefully. This project helped deepen my understanding of file handling, list comprehensions, and dictionary operations in Python.

Key Features

CSV Reading: Extracts data line-by-line and splits it based on commas. Dictionary-Based Storage: Stores each row as a dictionary with headers as keys for easy access. Error Handling: Uses try-except blocks to handle file not found errors and other I/O issues.
