Python Project 1 - CSV File Reader and Data Processing

This Python project is a CSV file reader and data processor developed as part of the CITS1003 unit at the University of Western Australia. Its primary function is to read data from CSV files, process it into a structured format using dictionaries and lists, and provide easy access to the data for further analysis.

Project Overview

The project focuses on:

Reading CSV Files: Uses the open() function to read CSV files and extract headers and data. Data Structuring: Converts each row into a dictionary using headers as keys. Error Handling: Implements try-except blocks to manage file reading errors gracefully. This project helped deepen my understanding of file handling, list comprehensions, and dictionary operations in Python.

Key Features

CSV Reading: Extracts data line-by-line and splits it based on commas. Dictionary-Based Storage: Stores each row as a dictionary with headers as keys for easy access. Error Handling: Uses try-except blocks to handle file not found errors and other I/O issues.

Python Project 2 - Social Media Usage Analysis

This Python project analyzes user demographics to understand social media usage patterns. It processes input data from CSV files and generates four outputs: OP1, OP2, OP3, and OP4 based on the analysis criteria. This project was developed as part of the CITS1003 unit at the University of Western Australia.

Project Overview

The project focuses on:

Data Reading and Parsing: Uses CSV file reading techniques to extract and parse user data. Data Analysis: Performs analysis to generate insights on user demographics and social media usage. Output Generation: Creates four outputs (OP1 to OP4) based on analysis results.

Key Features

CSV Reading: Extracts data line-by-line and stores it in lists for further analysis. Demographic Analysis: Analyzes data to identify trends based on age, gender, and other demographics. Custom Output Formats: Generates structured outputs (OP1, OP2, OP3, OP4) for reporting.
