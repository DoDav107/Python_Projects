Python Project 2 - Social Media Usage Analysis

This Python project analyzes user demographics to understand social media usage patterns. It processes input data from CSV files and generates four outputs: OP1, OP2, OP3, and OP4 based on the analysis criteria. This project was developed as part of the CITS1003 unit at the University of Western Australia.

Project Overview

The project focuses on:

Data Reading and Parsing:
Uses CSV file reading techniques to extract and parse user data.
Data Analysis:
Performs analysis to generate insights on user demographics and social media usage.
Output Generation:
Creates four outputs (OP1 to OP4) based on analysis results.

Key Features

CSV Reading:
Extracts data line-by-line and stores it in lists for further analysis.
Demographic Analysis:
Analyzes data to identify trends based on age, gender, and other demographics.
Custom Output Formats:
Generates structured outputs (OP1, OP2, OP3, OP4) for reporting.
